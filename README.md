# nanafy-site

The public marketing landing page for **Nanafy** — served at **https://nanafy.ai** via GitHub Pages.

- `index.html` — the whole site (single self-contained file).
- `CNAME` — tells GitHub Pages the custom domain is `nanafy.ai`.

To change the site: edit `index.html`, then commit and push. GitHub Pages redeploys automatically in about a minute.

This repo is intentionally separate from the private app repo (`willberry21/nanafy`), which holds the mobile app code.
